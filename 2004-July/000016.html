Alright, let's use this as a very basic example. I'm just going to
code the following below that has _nothing_ to do with the actual HTML
or PHP you will be using; it's just an example of how to use it.

<html>
<head>
<title>A PHP Example</title>
</head>
<body>

<!-- We will assume the following table is part of your template, a
nav bar, in a sense -->
  <table align="center" width="100%">
    <tr>
      <td>
       <a href="index.html">The Index</a>
      </td>
     </tr>
  </table>

   <!-- Now let's assume the next table is the main content of the site -->
   <table align="center" width="100%">
    <tr>
     <td>
      <p>
       <?php
          $somevariable = 2;
          if( $somevariable == 2) {
           print "This will be outputted to the browser.<br />";
          }
       ?>
      </p>
     </td>
    </tr>
   </table>
  </body>
</html>

Get it? If you still need some help, I could put it in for you. :)

Also, note that PHP print does not print directly to the browser. It
writes it to the HTML file. For example, <?php $paragraphit = "<p>";
$closeit = "</p>"; print $paragraphit; print "Test"; print $closeit;?>
would just print "Test" to the browser, but if you looked at the
source, it would say <p>Test</p>
