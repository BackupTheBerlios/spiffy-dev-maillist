The HTML should look something like:
<form method="POST" action="sendit.php">
 Name: <input type="text" name="name" /><br />
 Email: <input type="text" name="email" /><br />
 Please input a description of what you'd like to do for Spiffy and
your skills:<br />
 <textarea name="desc" rows="25" cols="10"></textarea>
</form>

The PHP part (sendit.php):
   <?php
     $error = 0; //If there is an error it gets set to one

/**************************************************************
** Be sure to se the variable $email_to below            **
**************************************************************/
     $email_to = "your email address"; // Set this to the email
address you want it sent to.

     //Make sure stuff exists
     if( $name == "" ) {
      print "Error: No name given.<br />";
      $error = 1;
     }
     if( $desc == "") {
      print "Error: No description given.<br />";
      $error = 1;
     }
     if( $email == "") {
      print "Error: No email address given.<br />";
      $error = 1;
     }
     if( $error != 0 ) {
      print "Script execution stopped due to errors.<br />";
      exit;
     }
     
     //Send mail
     $from = $email;
     $subject = "Forty-two Devloper: ".$name;
     $to = $email_to;
     $message = $desc;
     mail( $to, $subject, $message, $from )
      or $error = 1;
     if( $error != 0 ) {
      print "Error sending email. Please try again.<br />";
      exit;
     }
     
     //Tell the user about it
     print "An email has been sent to $email_to - below is a copy of
it. Thank you for joining the Spiffy team. You will be contacted
within a day.<br />";
     print "<br />";
     print "From: $email<br />";
     print "To: $email_to<br />";
     print "Subject: Forty-two Developer: $name<br />";
     print "<br />";
     print $desc."<br />";
   ?>

Make sure you include some HTML (like the template you're using for
the site) around sendit.php, too. Just don't put it inside the
<?php...?> tags. You may want to put the <?php...?> tags inside <p>
tags as it _will_ output stuff. :)
